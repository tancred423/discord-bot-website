M'naago is a Discord bot that can provide character profiles for the MMORPG Final Fantasy XIV (FFXIV, FF14), send automated Lodestone news notifications, display current maintenances, live letters and server statuses. The bot allows you to verify your character, view detailed character profiles with customizable themes, manage favorites, and stay up-to-date with the latest news, maintenances, and updates from the Lodestone.
