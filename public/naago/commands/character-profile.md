## /profile me

Get the profile of your verified character. Characters can be verified with `/verify add`.

## /profile find

Get any character's profile by providing the full character name and the server the character is on. Alternatively, you can right-click a user → `Apps` → `Find`. This is faster but requires this user to have a verified character.

## /profile favorite

Quick access to character profiles. Favorites can be added or removed with `/favorite add` and `/favorite remove`, respectively. Alternatively, you can right-click a user → `Apps` → `Add Favorite` / `Remove Favorite`. This is faster but requires this user to have a verified character.
