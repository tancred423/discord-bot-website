## /theme

Set a theme for your verified character's profile.

## /setup lodestone

Set up which channels receive automated Lodestone news updates. You can set or remove channels for each category of news. The Lodestone news categories are: Topic (Latest news, PLL's and patch notes), Notice (Secondary news), Maintenance (All kind of maintenances and their durations), Update (Outcome from maintenances), Status (Technical difficulties and server statuses). By default, this command requires `Manage Channels` permission to execute.

## /setup filters

Set up keyword filter blacklists to exclude certain Lodestone news. E.g. you can hide all "Ceremony of Eternal Bonding Reservation Restrictions" posts. By default, this command requires Manage Channels permission to execute.

## /setup event-reminders

Set up reminders for events that are ending soon and live letters that are starting.
