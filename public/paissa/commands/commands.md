## /paissa

Use this command to find housing information for a data center and world of your choice. You can further filter it within the command using the optional parameters:

* `[datacenter]` — The data center to search in
* `[world]` — The world to search in
* `[district]` — Housing District (Mist, The Lavender Beds, The Gauntlet, Shirogane, Empyreum)
* `[size]` — Plot Size (Small, Medium, Large)
* `[lottery-phase]` — Current Phase (Accepting Entries, Results, Unavailable, FCFS, Missing/Outdated)
* `[allowed-tenants]` — Who can enter the lottery (Free Company, Individual)
* `[plot]` — Specific plot number. Selects subdivision automatically (30 also shows 60)
* `[ward]` — Specific ward number
